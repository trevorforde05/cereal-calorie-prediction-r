# Nutritional Data Analysis & Calorie Prediction Model (R)

## Overview
Performed exploratory data analysis and regression modeling on cereal nutritional data to predict calorie values using sodium, fiber, carbohydrate, and sugar metrics.

## Techniques Used
- Exploratory Data Analysis (EDA)
- Correlation Testing
- ANOVA
- Multiple Linear Regression
- Stepwise Feature Selection
- Data Visualization

## Tools
- R
- ggplot2
- dplyr
